    <h1>Mortgage Loans Data Preprocessing</h1>
    <p>
        This project focuses on preprocessing a dataset of 150,000 mortgage loan applications from 2019. 
        The goal is to clean the data, handle missing values, and detect outliers to prepare it for predictive modeling and analysis.
    </p>

    <h2>Features</h2>
    <ul>
        <li><strong>Data Cleaning:</strong> Removed noisy and irrelevant data to enhance dataset quality.</li>
        <li><strong>Handling Missing Values:</strong> Applied imputation techniques to address incomplete records.</li>
        <li><strong>Outlier Detection:</strong> Identified and treated outliers to ensure accurate modeling.</li>
        <li><strong>Visualization:</strong> Leveraged Matplotlib for data exploration and insights.</li>
    </ul>

    <h2>Libraries Used</h2>
    <ul>
        <li><strong>Pandas:</strong> For data manipulation and analysis.</li>
        <li><strong>NumPy:</strong> For numerical computations.</li>
        <li><strong>Matplotlib:</strong> For data visualization.</li>
    </ul>

    </ol>
